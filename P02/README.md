# 📘 README — Anàlisi de Llicenciament Windows Server 2025 (TransLògic S.A.)

## Introducció

Mentre arrenca la vostra etapa com a empresa, EverPia us deriva encàrrecs de clients.  
El primer és **TransLògic S.A.**, una empresa de logística que vol renovar la seva infraestructura i virtualitzar tots els seus serveis a **Windows Server 2025**.

Actualment tenen un servidor físic antic i volen migrar a un entorn virtual complet.

## Detalls de la infraestructura del client

### 🖥️ Servidor físic (host)

*   2 CPUs
*   Cada CPU té 12 nuclis
*   **Total: 24 cores**

### 🧱 Càrrega de treball (12 VMs Windows Server)

*   1 DC (Active Directory)
*   1 Servidor de fitxers
*   1 Servidor d’impressores i documental
*   1 SQL Server (ERP)
*   8 VMs de suport (logística + terminals de magatzem)

### 👥 Usuaris i dispositius

*   45 empleats
*   30 treballadors d’oficina (PC + portàtil)
*   15 mossos (5 tauletes en torns)

# 🧩 Encàrrec del client

L’objectiu és presentar una proposta clara, econòmica i ben justificada.

Haureu de:

1.  **Analitzar el model de llicenciament per core** de Windows Server 2025.
2.  **Calcular el cost total** amb les dues edicions principals:
    *   *Windows Server Standard*
    *   *Windows Server Datacenter*
3.  Determinar si surt més econòmic utilitzar **CALs per Usuari** o **CALs per Dispositiu**.
4.  Justificar la decisió final tenint en compte:
    *   Cost
    *   Escalabilitat
    *   Nombre de VMs
    *   Funcionalitats avançades (Storage Spaces Direct, SDN, etc.)
5.  Preparar una **presentació clara per al gerent**, enfocada a una persona no tècnica (5–10 min).
