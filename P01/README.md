# 📘 README — Treball amb Git, GitHub i GitHub Classroom

## Introducció

Fins ara heu lliurat la documentació directament des de la web de GitHub. Tot i que funciona, és lent, sobretot quan cal crear carpetes, pujar imatges o organitzar fitxers.  
Per millorar la productivitat i treballar com es fa en entorns reals, a partir d’aquest projecte fareu servir **git en local** i sincronitzareu la feina amb GitHub.

A més, solucionareu un problema important: fins ara els repositoris eren **públics**, i això fa impossible controlar si algú copia la vostra feina. Per això utilitzarem **repositoris privats**.

## GitHub Classroom

Treballarem amb **GitHub Classroom**, una plataforma pensada per entorns educatius.  
Permet tenir una estructura igual a la d’una organització real:

*   L’organització és propietària dels repositoris.
*   Cada alumne rep un **fork automàtic** i **privat** del repositori base.
*   Només vosaltres i el professorat teniu accés al vostre repo.

Més endavant, aquests repositoris es faran **públics**, així els podreu copiar al vostre GitHub personal i afegir-los al portafoli professional.
