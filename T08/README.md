# 📘 README — Formació en Seguretat contra Malware

## Introducció

La seguretat davant el malware és una de les parts més crítiques de l’administració de sistemes. Qualsevol tècnic ha de saber identificar riscos, protegir equips i entendre com funcionen les principals amenaces.  
Per millorar aquestes competències, realitzareu una formació específica que després podreu oferir com a **servei d’assessoria de seguretat**.  
Aquesta primera part se centra en el **malware** i en el seu impacte real dins d’una organització.

## Contingut de la formació

Treballareu amb diferents mecanismes de defensa disponibles a **Windows**:

*   Antivirus i eines d’anàlisi
*   Proteccions integrades del sistema
*   Configuracions segures
*   Estratègies de detecció i resposta

També observareu l’impacte real de determinats programes maliciosos en un entorn controlat per entendre **què fan, com actuen i per què són una amenaça**.

Aquesta activitat combina teoria, pràctica guiada i anàlisi crítica, amb l’objectiu de desenvolupar criteri professional a l’hora de protegir sistemes i xarxes.
